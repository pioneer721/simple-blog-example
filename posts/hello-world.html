Hello, world

April 23, 2026 · 2 min read

This is the first post on the blog. I wanted something lightweight. No framework, no build step, no database. Just HTML and CSS, hosted as static files.

The rule of thumb I follow: every post lives in its own file under the posts/ folder. The home page lists the posts, and each card links to the corresponding file. That way the home page stays short, and every post has a clean, shareable URL.

Why I like this setup

  • Every page loads fast because it is a plain HTML file.
  • I can open any post in a browser locally to preview it.
  • Hosting is free on GitHub Pages.
  • There is nothing to break. No runtime, no server.

More posts will follow as I get time.
